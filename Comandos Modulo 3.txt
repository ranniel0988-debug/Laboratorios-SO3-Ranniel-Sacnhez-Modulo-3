Practica 2:

Crear un script que haga un backup creando un comprimido utilizando TAR de la carpeta /home/(tu_usuario). el nombre del archivo debe de tener la fecha y la hora actual en formato dd-mm-yyyy:hh:mm
Crear un script que copie el resultado del comando ifconfig en un archivo en el escritorio. El script debe de preguntar cual nombre desea ponerle al archivo y crear el archivo con el nombre provisto
==================================
nano backup.sh
#dentro de este archivo pegar o crear el script

#!/bin/bash

USUARIO=$(whoami)

FECHA=$(date +"%d-%m-%Y_%H.%M")
NOMBRE_ARCHIVO="backup_${USUARIO}_${FECHA}.tar.gz"

tar -pczf "$NOMBRE_ARCHIVO" "/home/$USUARIO"

echo "Backup creado exitosamente: $NOMBRE_ARCHIVO"

#dar permisos de ejecucion
sudo chmod +x backup.sh
# ejecutar el script
./backup.sh

nano ifconfig.sh
#!/bin/bash

echo "Introduce el nombre que deseas para el archivo (sin extensión):"
read NOMBRE_USUARIO

RUTA_ESCRITORIO="$HOME/Desktop"

ifconfig > "${RUTA_ESCRITORIO}/${NOMBRE_USUARIO}.txt"

echo "El resultado de ifconfig se ha guardado en: ${RUTA_ESCRITORIO}/${NOMBRE_USUARIO}.txt"

sudo chmod +x ifconfig.sh
./ifconfig.sh
history
